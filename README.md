# ACA Schools - Auchi College of Achievement

> **"From Ruins to Renaissance"** — The digital monument of a legendary comeback.

## Project Structure

```
├── app/                    # Next.js app directory
├── components/             # Reusable React components
├── pages/                  # Route pages
├── public/                 # Static assets
├── styles/                 # Global CSS
└── lib/                    # Utilities and data
```

## Pages

1. **Homepage** - Cinematic hero with impact stats
2. **Resurrection Story** - The comeback narrative
3. **Facilities** - ICT Center, Labs, Sports, Infrastructure
4. **Academics** - Junior & Senior Secondary
5. **Admissions** - Simple application form
6. **Alumni Platform** - Directory & networking
7. **Gallery** - Before/After, Events, Media
8. **News** - Blog and announcements
9. **Admin Dashboard** - Content management (simple)

## Tech Stack

- **Framework**: Next.js 14 (React)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Render (Free)

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment to Render

1. Push to GitHub
2. Connect to Render
3. Set build command: `npm run build`
4. Set start command: `npm start`
